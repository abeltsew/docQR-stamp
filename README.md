## 💻 Getting Started <a name="getting-started"></a>

> This project is made using react with vite

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to follow the below steps

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/abeltsew/docqr-stamp.git
```

-

### Install

Install this project with:

npm install or yarn

### Usage

To run the project, execute the following command:


start the project with:

```sh
 npm run dev
```

or

```sh
yarn dev
```


### Deployment

You can deploy this project using netlify or vercel
